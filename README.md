# Système de prédiction du défaut de paiement

## Contexte
Ce projet a pour objectif de développer un système de prédiction du défaut de paiement
à partir de données clients liées aux demandes de crédit.

Le défaut de paiement représente un enjeu important pour les institutions financières,
car une mauvaise évaluation du risque peut entraîner des pertes économiques.
Ce projet vise à exploiter les données disponibles afin d’aider à la prise de décision
lors de l’octroi d’un crédit.

Ce travail est réalisé dans le cadre d’un projet académique.

---

## Problématique
Comment prédire, à partir des informations financières et socio-démographiques d’un client,
le risque de défaut de paiement ?

---

## Objectif du projet
Mettre en place un système de prédiction capable d’identifier les clients présentant
un risque de défaut de paiement à partir de leurs données.

---

## Objectifs spécifiques
- Analyser et comprendre un jeu de données réel
- Préparer et nettoyer les données
- Identifier les variables pertinentes
- Construire des modèles de prédiction
- Évaluer et analyser les résultats obtenus

---

## Jeu de données
Le projet repose sur un jeu de données réel contenant des informations anonymisées
relatives à des clients ayant effectué une demande de crédit.

Pour des raisons de taille, le jeu de données est conservé localement
et n’est pas inclus dans le dépôt GitHub.

---

## Méthodologie
Le projet suit une démarche structurée :
1. Exploration et compréhension des données
2. Prétraitement et nettoyage
3. Modélisation
4. Évaluation des performances
5. Interprétation des résultats
